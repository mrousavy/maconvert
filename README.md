# maconvert

Easily convert `.svg` files to `.png` files.

<div align="center">
    <img src="img/demo.png" alt="demo picture" />
</div>

```sh
$ npm install -g maconvert
```

### Usage

```sh
$ maconvert input.svg
# or
$ svgpng input.svg
```
